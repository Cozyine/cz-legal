# Cozine | Legal & Documentation

Official repository for **Cozine**, a security and anti-exploit utility for Discord servers.

## Documentation
*   **[Terms of Service & Privacy Policy](https://cozyine.github.io/cz-legal/)**

## About the Bot
Cozine helps moderators find and flag exploiters before they can cause issues. We use a global database to keep communities safe from raids and "script kiddies".

## Data & Privacy
We only store the essentials: User IDs, Guild IDs/Names and Roblox Information.
*   **Retention**: To keep the database effective, security logs aren't deleted for users who break the rules.
*   **Appeals**: If you think you were flagged by mistake, join the support server below to file an appeal.
